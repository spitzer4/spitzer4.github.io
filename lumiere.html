Lumiere

This case study details the design and development of a static website for Lumiere Fragrance, a fictional luxury fragrance brand. The project aimed to create a visually captivating and user-friendly online experience that would showcase the brand's signature scents, embody its commitment to quality and craftsmanship, and ultimately drive customer engagement. My role encompassed both UX/UI design and front-end web development, allowing for a cohesive and integrated approach.

Project Goals

  • Develop a visually stunning website that reflects the brand's luxurious image.
  • Create a user-friendly experience for browsing signature scents.
  • Effectively communicate the brand's story and values.
  • Implement a responsive design that adapts to various screen sizes.
  • Deliver a clean and well-documented codebase for future maintenance and updates.

UX/UI Design Process

  1. Brand Research and Inspiration: I began by immersing myself in the world of luxury fragrances, researching competitor websites and identifying key design elements that convey elegance and sophistication. This research informed the visual direction and overall aesthetic of the website.
  2. User Flow and Information Architecture: I mapped out the user journey, focusing on how visitors would navigate the site to discover and explore the fragrance collection. A clear information architecture was crucial for ensuring a smooth and intuitive browsing experience. (***Visuals like a simplified sitemap or user flow diagram would be placed here in a real portfolio).
  3. Wireframing and Prototyping: Low-fidelity wireframes were created to outline the website's layout and structure, focusing on usability and information hierarchy. These wireframes were then translated into high-fidelity mockups, showcasing the visual design, typography, and color palette.
  4. Visual Design and Branding: The visual design was carefully crafted to reflect Lumiere's brand identity. A sophisticated color palette, elegant typography, and high-quality imagery were selected to create a sense of luxury and exclusivity.

Web Development

The website was built using HTML for structure and CSS for styling, adhering to best practices for clean code and maintainability.

  • HTML Structure: Semantic HTML5 elements were used to create a clear and logical structure for the website's content, improving accessibility and SEO.
  • CSS Styling: CSS was used to style all aspects of the website's appearance, including typography, colors, layout, and responsiveness. Flexbox was utilized for flexible and efficient layout design. A mobile-first approach was considered to ensure optimal viewing on smaller screens.
  • Image Optimization: Images were optimized for web use to reduce file sizes without sacrificing quality, improving page load times.

Design & Development Integration

The UX/UI design and web development phases were closely integrated. The design decisions made during the UX phase directly informed the development process. For example, the elegant typography and color palette were implemented using CSS, and the layout defined in the wireframes was brought to life using HTML and Flexbox.

Challenges & Solutions

  • Challenge: Creating a visually engaging scent slider with only HTML and CSS.
  • Solution: Leveraged CSS properties like overflow-x: auto and scroll-snap to create a horizontally scrolling container for the scent cards. While basic, this provides the desired functionality without JavaScript.

Results

  • A visually appealing and user-friendly website that effectively showcases Lumiere Fragrance's brand and products.
  • A clean and well-structured codebase that is easy to maintain and update.
  • A functional scent showcase using only HTML and CSS.

Conclusion

This project successfully demonstrated the ability to create a professional-looking website using only HTML and CSS. While the absence of JavaScript and media queries presented some limitations in terms of advanced interactivity and fully responsive design, the project achieved its core goals of visual appeal, user-friendliness, and effective brand communication. This case study highlights the importance of semantic HTML, well-structured CSS, careful planning, and the synergy between UX/UI design and web development in creating a successful online presence. For future iterations, incorporating JavaScript and media queries would significantly enhance the site's functionality and responsiveness.
